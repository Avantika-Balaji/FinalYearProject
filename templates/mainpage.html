Shot Boundary Detection with Video Caption

OVERVIEW

Automatic identification of shot transitions in a video is a classical task of video analysis, and as the benchmarks do not cover all “open-world” aspects of video content, the problem cannot be considered as solved yet. Besides false hits, trained models can sometimes miss a difficult (e.g. long) transition, or may find shot transition detection models “unprepared” for a given type. Thus, it is underlined that there is a need for an elegant and efficient system for detecting transitions that makes video segmentation easier and more precise.

This is to be aided through a captioning model that returns a set of natural language descriptions for a given segment of a video. Ultimately, the goal is to input a video and get as an output, the points of Shot Transitions and along with that a set of tags pointing to what has taken place in between the said transitions. Finally, this is illustrated in a simple graphical interface.
